# ai-learning

## Core Foundations (must-know)
- Mathematics: Linear algebra; multivariable calculus; probability & statistics; optimization.
- Programming & Tools: Python; NumPy; pandas; Git; Docker; Linux basics.
- Data Engineering: Data cleaning; feature engineering; ETL; databases; data versioning.

## Machine Learning (ML) topics
- Supervised learning: Linear/logistic regression; decision trees; SVMs; ensemble methods (Random Forest, XGBoost).
- Unsupervised learning: Clustering (k‑means, hierarchical); dimensionality reduction (PCA, t‑SNE, UMAP).
- Probabilistic models: Bayesian inference; graphical models; HMMs.
- Model evaluation: Cross‑validation; ROC/AUC; calibration; bias‑variance tradeoff.
- Feature engineering & selection.

## Deep Learning (DL) topics
- Neural network basics: MLPs; activation functions; loss functions; backpropagation.
- Architectures: CNNs (vision); RNNs/LSTMs (sequences); Transformers (attention).
- Training at scale: Batch norm; optimizers (SGD, Adam); learning rate schedules; regularization.
- Generative models: VAEs; GANs; diffusion models.
- Representation learning & transfer learning.
